📘 Frequently Asked Questions

To update your office hours:

  1. Go to 'Settings' in the sidebar
  2. Click on 'Profile Settings'
  3. Scroll to the 'Office Hours' section
  4. Update your availability
  5. Click 'Save Changes'

Your updated office hours will be visible to all students enrolled in your courses.

To send a message to all students in your course:

  1. Navigate to 'Courses' in the sidebar
  2. Select the specific course
  3. Click on the 'Announcements' tab
  4. Select 'New Announcement'
  5. Compose your message
  6. Toggle 'Send as email' if you want students to receive an email copy
  7. Click 'Post Announcement'

All enrolled students will receive the announcement on their dashboard and via email if selected.

To add a teaching assistant:

  1. Go to 'Courses' in the sidebar
  2. Select the course you want to add a TA to
  3. Click on 'Course Management'
  4. Select 'Teaching Staff'
  5. Click 'Add Teaching Assistant'
  6. Enter the email address of the TA
  7. Set appropriate permissions
  8. Click 'Send Invitation'

The TA will receive an email invitation to join your course with the specified permissions.

To copy content from a previous course:

  1. Go to 'Courses' in the sidebar
  2. Click 'Create New Course'
  3. Fill in the basic course information
  4. At the bottom, select 'Import from Existing Course'
  5. Choose the source course from the dropdown
  6. Select which elements to import (assignments, quizzes, materials, etc.)
  7. Click 'Create Course'

All selected content will be copied to your new course with dates adjusted relative to the new start date.

To upload multiple files efficiently:

  1. Navigate to your course
  2. Go to 'Course Materials' tab
  3. Click 'Add Materials'
  4. Select 'Bulk Upload'
  5. Drag and drop multiple files or click to browse
  6. Organize files into appropriate modules/folders
  7. Click 'Upload All Files'

You can upload up to 100MB at once. For larger files, use the course file server option.

To create student groups for team projects:

  1. Go to your course page
  2. Select 'Course Management'
  3. Click on 'Groups'
  4. Select 'Create Group Set'
  5. Configure group options (size, self-enrollment, etc.)
  6. Choose to create groups manually or automatically
  7. Set group permissions and options
  8. Click 'Create Groups'

You can later assign specific assignments to be completed by groups rather than individuals.

To create a grading rubric:

  1. Go to 'Courses' and select your course
  2. Click on 'Assignments' or 'Assessments'
  3. Select the assignment you want to add a rubric to
  4. Click 'Edit' and scroll to 'Rubric'
  5. Click 'Add Rubric'
  6. Define criteria, performance levels, and point values
  7. Save the rubric

You can also save rubrics as templates to reuse across multiple assignments.

To leave audio feedback:

  1. Open the assignment submission you're grading
  2. In the feedback section, click the microphone icon
  3. Record your feedback (up to 3 minutes)
  4. Preview the recording
  5. Click 'Save' to attach the audio to your feedback

Audio feedback is often appreciated by students as it provides more context and feels more personal than written comments.

To export your gradebook:

  1. Navigate to 'Grades' in the sidebar
  2. Select the course
  3. Click the 'Export' button in the top-right
  4. Choose your export format (Excel, CSV, etc.)
  5. Select what data to include
  6. Click 'Download'

Note that any changes made to the downloaded file will not automatically sync back to EduConnect. To import updated grades, use the 'Import' feature.

EduConnect supports the following file types:

  • Documents: PDF, DOC, DOCX, TXT, RTF, ODT
  • Spreadsheets: XLS, XLSX, CSV, ODS
  • Presentations: PPT, PPTX, ODP
  • Images: JPG, JPEG, PNG, GIF, SVG
  • Audio: MP3, WAV, M4A (max 500MB)
  • Video: MP4, MOV, AVI (max 2GB)
  • Archives: ZIP, RAR (max 1GB)
  • Other: HTML, XML

Note: Files over 2GB should be shared via external link using the 'Add External Resource' option.

To set up video conferencing:

  1. Go to 'Settings' in the sidebar
  2. Select 'Integrations'
  3. Connect your preferred video conferencing tool (Zoom, Teams, etc.)
  4. Grant necessary permissions
  5. Go to your course page
  6. Select 'Virtual Meetings'
  7. Click 'Schedule Meeting' and configure settings
  8. Save and publish the meeting link

Students will receive notifications and can join directly through the course page.

To ensure your course displays well on mobile devices:

  • Use the 'Mobile Preview' tool in the course editor to see how content appears
  • Keep file sizes reasonable (compress images before uploading)
  • Break long content into smaller chunks with clear headings
  • Use mobile-friendly file formats (MP4 instead of MOV, PDF instead of PPTX)
  • Test interactive elements on a mobile device
  • Consider providing alternative formats for complex tables or diagrams

Approximately 40% of students access course materials primarily via mobile devices, so mobile optimization is increasingly important.

🔧 Common Troubleshooting

Gradebook Not Calculating Correctly

  1. Check category weightings add up to 100%
  2. Verify all assignments have correct point values
  3. Ensure "Exclude from Grade Calculation" is not checked
  4. Look for hidden assignments affecting totals

Students Can't See Course Content

  1. Check if content is set to "Published"
  2. Verify release date and time settings
  3. Confirm students are properly enrolled
  4. Check if prerequisite conditions are set

Quiz Timer Issues

  1. Verify time limit is correctly set
  2. Check if "Auto-submit" is enabled
  3. Review settings for extended time accommodations
  4. Ensure browser compatibility for all students

File Upload Errors

  1. Check file size is under limit (2GB max)
  2. Verify file type is supported
  3. Try compressing or converting the file
  4. Clear browser cache or try a different browser
